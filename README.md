📊 E-Commerce Data Analysis – Power BI Project
Project Overview
This project analyses retail sales data using Power BI to evaluate business performance, profitability, regional trends, and the impact of discounts. The goal is to generate actionable business insights through interactive dashboards.

Key Objectives
•	Analyses overall sales and profit performance
•	Identify high-performing regions and product categories
•	Evaluate discount impact on profit margins
•	Provide data-driven business recommendations

Key KPIs
•	Total Sales
•	Total Profit
•	Profit Margin %
•	Total Quantity Sold
•	Year-over-Year Growth

Report Pages
•	Index / Navigation Page
•	Executive Overview
•	Sales Analysis
•	Profit Analysis
•	Discount & Quantity Analysis
•	Final Insights & Recommendations

Tools Used
Microsoft Power BI, DAX
